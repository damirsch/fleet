## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```![localhost_3000_](https://github.com/damirsch08/fleet/assets/97800728/761032a4-7891-4072-ae37-8036cb05d9ae)
